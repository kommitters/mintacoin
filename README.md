# Mint a Coin

Mint a Coin is a REST API for easily dealing with cryptoassets helping to abstract away the complexity and simplify these arduous processes.


## Installation

1. Install dependencies
```bash
mix deps.get
```

2. Create and migrate database
```bash
mix ecto.setup
```

3. Start server
```bash
mix phx.server
```


## Run tests

```bash
mix test
```
